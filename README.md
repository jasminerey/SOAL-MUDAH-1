Penjelasan singkat tentang fungsi dan cara kerja program:
- Dalam program ini saya menggunakan Scanner agar bisa mengimput data dari pengguna.
- Program ini meminta pengguna untuk mengimput volume paket yang disimpan di variabel panjang, lebar, dan tinggi
- Program ini menggunakan logika if else untuk menentukan biaya tambahan (jika volume paket > 100 akan terkena biaya tambahan 5000) dan biaya per kg (jika jarak tempuh <= 10km maka biaya 4250, jika jarak tempuh > 10 maka biaya 6000)
